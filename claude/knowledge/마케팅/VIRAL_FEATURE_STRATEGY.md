## 0. 5단계 성장 엔진 매핑

> 이 문서의 모든 섹션은 [범용 성장 공식](../비즈니스/UNIVERSAL_GROWTH_FORMULA.md)의 5단계 엔진에 매핑된다.
> 깊은 원리가 필요하면 해당 문서의 Stage를 참조하라.

```
① 본능 → ② 훅 → ③ 확산 → ④ 전환 → ⑤ 잠금 → ①로 복귀 (플라이휠)
```

| 본 문서 섹션 | 성장 엔진 Stage | 연결 포인트 |
|---|---|---|
| **§1. 설계 원칙** | ② 훅 + ③ 확산 | 트렌드 훅 → 유입 → 공유 루프 |
| **§2. 기능 로드맵** | ②③④ 전체 | Tier별 기능이 훅→확산→전환을 순차 설계 |
| **§3. 후킹 패턴** | ② 훅 | 8대 훅 전략의 SNS 실전 버전 (숫자 권위, 손실 회피, FOMO 등) |
| **§4. 플랫폼 관심도** | ① 본능 | 어떤 본능(시기, 불안, 음욕)이 어느 플랫폼에서 강한지 |
| **§5. 데이터 수집 도구** | ③ 확산 | 바이럴 삼각형의 "차별화 소재" 발굴용 |
| **§6. 트렌드 모니터링** | ① 본능 + ② 훅 | 본능이 표출되는 시의성 포착 |
| **§7~8. 케이스 스터디** | ①②③ 전체 | 각 사례가 어떤 본능 조합 × 훅 × 확산 구조를 쓰는지 |
| **§9. 5가지 심리 트리거** | ① 본능 | 7대 본능의 바이럴 특화 축약 버전 |
| **§10. 욕망 트리거별 분류** | ① 본능 + ③ 확산 | 본능별 바이럴 삼각형 실전 사례 |
| **§11. 트리거별 공식 정리** | ①③ 통합 | 본능 → 확산 메커니즘 1:1 매핑 |
| **§12. 기능 티어 분류** | ②③④ 전체 | S/A/B 티어 = 훅+확산+전환 강도순 |
| **§13. 핵심 인사이트** | ①~⑤ 전체 | 12개 인사이트가 5단계 전체에 분산 |

**빠진 Stage 주의**:
- **④ 전환**: 이 문서는 CTA 수준만 다룸 → 상품 사다리·PAS·가격 전략은 [성장 공식 Stage 4](../비즈니스/UNIVERSAL_GROWTH_FORMULA.md#stage-4-전환-설계--왜-돈을-내는가) 참조

## 1. 바이럴 기능 설계 원칙

### 토스에서 배우는 공식

> **시의성 있는 트렌드 훅 → 유저 유입 → 핵심 서비스로 자연스러운 연결**

토스 '두쫀쿠맵' 사례:
- 1인 개발자가 만든 재고 확인 서비스 → 토스 미니앱으로 **하루 만에 출시**
- 한 달 만에 **MAU 108만** 달성
- 두쫀쿠 재고 확인 → 페이스페이 결제 유도 (할인 쿠폰)
- 출처: https://toss.im/apps-in-toss/blog/dubaicookiemap

### 범용 적용 공식

> **공감/호기심 트리거 → 진단/결과 생성 → 예쁜 카드로 공유 → 유료 서비스 전환**

이 공식은 운세, 뷰티, 교육, F&B, SaaS 등 분야를 가리지 않는다. 핵심은 **결과물이 공유하고 싶을 만큼 매력적**이어야 한다는 것.

## 2. 우선순위별 바이럴 기능 설계 프레임워크

### Tier 1: 즉시 실행 가능 (난이도 낮음, 효과 높음)

#### A. 결과 공유 카드 강화 (스포티파이 Wrapped 스타일)

**벤치마크**: 스포티파이 Wrapped, W컨셉 심리테스트 (2주 18만 플레이)

어떤 서비스든 결과 공유 기능이 있다면, 바이럴을 위해 필요한 것:
- 인스타 스토리에 최적화된 **9:16 세로 카드** 이미지
- "나의 유형은 ___" 한 줄 + 시각적 아이콘
- 공유 시 자동으로 **서비스 URL** 링크 포함
- 계절/시즌별 디자인 변경 (벚꽃, 크리스마스 등)

**핵심 지표**: 공유 카드 생성 수, SNS 유입 트래픽

#### B. 화제 콘텐츠 시의성 연동

**벤치마크**: 토스 두쫀쿠맵 (트렌드 속도 = 생명)

설계 원칙:
- 자기 분야와 관련된 화제의 인물/사건/트렌드를 포착
- 해당 트렌드를 자사 서비스의 프레임으로 재해석한 콘텐츠 발행
- **핵심**: 트렌드 감지 후 **24시간 이내 콘텐츠 발행**이 관건

### Tier 2: 2~4주 개발 (난이도 중간, 효과 매우 높음)

#### C. 심리테스트형 퀴즈

**벤치마크**: W컨셉 심리테스트 (2주 18만), 각종 MBTI 테스트

구조:
1. 5~8개 질문 (감정 기반, 빠른 진행)
2. 결과: 자사 프레임워크 기반 유형 분류 (예: "불꽃형 리더십")
3. 궁합/매칭: "나와 잘 맞는 유형은 ___"
4. 공유 카드 자동 생성
5. CTA: "더 자세한 분석 보기" → 유료 전환

**SNS에서 검증된 주제 유형**:
- 자가진단 리스트형 ("~하는 사람 특징 N가지")
- 애착 유형/성격 분류형
- 구조 분석형 ("시장에서의 내 포지션")

#### D. 연간/반기 리포트 자동 생성

**벤치마크**: 스포티파이 Wrapped, 카카오 이모티콘 연말결산

구조:
- "2026년 상반기 나의 OOO 리포트"
- 6장짜리 카드 시퀀스 (카테고리별 핵심 인사이트)
- 각 카드 개별 공유 가능
- 연말/반기에 시즌 한정 발행 → 희소성

### Tier 3: 장기 검토 (난이도 높음, 확장성 큼)

#### E. 슈퍼앱 미니앱 입점

- 카카오톡/토스 등 슈퍼앱 미니앱으로 입점
- 수천만 유저 노출 + 로그인 허들 제거
- 푸시 알림으로 재방문 유도

#### F. AI 챗봇 연동 (SNS DM)

- DM으로 정보 입력 → AI 분석/상담
- 결과를 스토리로 공유하면 추가 분석 제공
- 팬덤/커뮤니티 연계 가능

## 3. SNS 인기글 후킹 패턴 (콘텐츠 기획 참고)

### 검증된 후킹 공식 4가지

| 패턴 | 공식 | 적용 예시 |
|------|------|----------|
| **숫자 권위** | "N명 상담해봄 → 깨달은 것" | "AI가 10만명 분석한 패턴" |
| **자가진단 리스트** | "~하는 사람 특징 N가지" | "올해 성공하는 사람 특징 5가지" |
| **통념 뒤집기** | "A인 줄 알지? 사실 B임" | "MBTI보다 정확한 성격 분석 방법" |
| **한 줄 팩폭** | 짧고 단정적 문장 | "오늘의 한 줄 인사이트" 시리즈 |

### 최강 조합

> **숫자 권위 + 리스트형 + 마지막에 질문**
> → 댓글 유도 + 공유 + 저장 동시 달성

## 7. 바이럴 케이스 스터디 (해외)

### A. 16Personalities / MBTI Chemistry Quiz
- **개요**: 16가지 성격유형 테스트 + 궁합 비교
- **바이럴 원리**: 정체성 표현("나는 이런 사람") + 2인 루프(상대 MBTI 필요 → 테스트 유도)
- **공유 메커니즘**: 결과 카드 스크린샷 → 인스타 스토리/틱톡/트위터 바이오
- **규모**: 누적 10억+ 테스트, 월 1,590만 방문
- **시사점**: 자사 프레임워크 기반 유형 카드를 MBTI처럼 "한 장짜리 정체성 카드"로 만들면 동일 효과

### B. "How Well Do You Know Me?" 퀴즈 (BuddyMeter, FriendQuiz)
- **개요**: 내가 10개 질문에 답 → 링크 공유 → 친구가 맞추기 → 점수 리더보드
- **바이럴 원리**: **제품 자체가 공유 없이 작동 불가** (강제 바이럴 루프)
- **공유 메커니즘**: 고유 URL 생성 → 카톡/인스타 스토리로 전파 → 친구가 자기 퀴즈도 생성 → 체인 반응
- **규모**: TikTok #FriendshipQuiz 5억+ 조회
- **시사점**: "나를 얼마나 아는지" 퀴즈 → 친구에게 링크 공유 → 친구도 서비스 유입

### C. 36 Questions to Fall in Love (NYT)
- **개요**: 심리학 연구 기반, 점점 깊어지는 36개 질문으로 친밀감 형성
- **바이럴 원리**: 저항 불가 헤드라인("누구와든 사랑에 빠지는 법") + 실제 연구 기반 신뢰도 + 체험자 후기
- **규모**: NYT 2015년 최다 공유 기사 중 하나, 10년 지나도 여전히 회자
- **시사점**: 학술/연구 기반 콘텐츠는 신뢰감으로 장기적 바이럴 가능

### D. Bird Test (틱톡)
- **개요**: 파트너에게 "오늘 새 봤어" 같은 사소한 말을 하고 반응 촬영. Gottman의 "관계 신호" 연구 기반
- **바이럴 원리**: 누구나 즉시 실행 가능 + 합격/불합격 긴장감 + 심리학 근거
- **규모**: 원본 영상 5,600만 조회
- **시사점**: 간단한 "테스트/챌린지" 포맷은 참여 장벽이 낮아 바이럴 폭발력이 높음

### E. Love Language Test (5가지 사랑의 언어)
- **개요**: 신체 접촉/칭찬/봉사/시간/선물 중 나의 사랑 언어 진단

### F. Dating Wrapped (스포티파이 Wrapped 패러디)

- **바이럴 원리**: 첫 테스트 "나에게 맞는 대학 스타일" → 네이버 실검 1위
- **규모**: MAU 300만+, 누적 1,100만 유저. **완료율 92%**, CTR 30~40%
- **시사점**: Smore 같은 플랫폼에 자사 퀴즈를 올리는 것도 유효한 채널 전략

### E. 이름궁합 테스트
- **개요**: 두 사람 이름 획수로 궁합 퍼센티지 계산 (지락실, 봉봉, 심콩 등)
- **바이럴 원리**: **3초면 끝남**(이름 2개 입력) + 숫자 결과 → "우리 94%래!" → 대화 소재
- **공유 메커니즘**: 결과 스크린샷 → 카톡/인스타 DM. 친구들이 자기 이름+짝사랑 테스트 → 체인
- **시사점**: 핵심은 **결과를 공유하고 싶게 만드는 카드 디자인** + **초저마찰 입력**

### F. 밸런스 게임
- **개요**: "코 고는 애인 vs 잠꼬대하는 애인" 양자택일 연애 질문
- **바이럴 원리**: 정답 없음 → 논쟁 유발 → 댓글/대화 폭발. 마라맛(극단적) 버전이 더 바이럴
- **공유 메커니즘**: 커플/친구가 플레이하는 영상 → 틱톡/인스타. 리액션이 콘텐츠
- **시사점**: 자사 프레임워크별 맞춤 질문 세트로 차별화 가능

## 10. 바이럴 케이스 스터디: 욕망 트리거별 분류

### 🔴 비교/랭킹 ("나는 몇 점이지?")

> 사람은 자신의 절대적 가치보다 **남과 비교한 상대적 위치**에 집착한다.
> 점수·퍼센타일·등급을 주면, 높으면 자랑하고 낮으면 자조하며 — **어느 쪽이든 공유한다.**

#### A. Tinder 히든 ELO 스코어
- **개요**: 틴더는 내부적으로 모든 유저에게 체스 ELO 방식의 "매력도 점수"를 매기고, 비슷한 등급끼리만 서로 노출시켰다. 유저는 이 점수를 볼 수 없었지만, Fast Company가 이를 폭로하면서 "내 점수가 몇 점이지?"라는 공포가 글로벌 미디어를 탔다.
- **바이럴 원리**: 숨겨진 평가 시스템의 존재 → "나도 모르게 등급이 매겨지고 있었다" 충격 → 기사/SNS 공유 폭발
- **규모**: 틴더 MAU 7,500만 시점에 글로벌 미디어 보도
- **적용법**: 숨겨진 등급을 "공개"하는 형태 — "당신의 OOO 점수는 상위 15%" 퍼센타일 출력

#### B. Photofeeler (사진 평가 플랫폼)
- **개요**: 프로필 사진을 업로드하면, 낯선 사람들이 '매력', '신뢰감', '지적 인상' 3가지 항목을 퍼센타일로 평가해준다. 원래 링크드인/데이팅 프로필 최적화 용도였지만, "나 진짜 몇 점이야?"라는 궁금증에 일반 유저까지 쏟아졌다.
- **바이럴 원리**: 모호한 자기 인식에 **정확한 숫자**를 붙여줌 → 결과를 안 볼 수 없음
- **규모**: Time, Forbes, Today Show 소개. 수백만 평가 처리
- **적용법**: 어떤 분야든 "나의 OOO 점수" 포맷으로 퍼센타일 출력 → 공유 욕구 자극

#### C. AI 매력 점수 필터 (PrettyScale, FaceRate, Hotness.ai)
- **개요**: 셀카를 업로드하면 AI가 얼굴 대칭·비율 등을 분석해 미모 점수를 1~10점으로 산출. 틱톡에서 "AI가 나한테 몇 점 줬게?"라는 리액션 영상이 #attractivenessscale로 수천만 조회를 기록했다.
- **바이럴 원리**: 숫자 결과 → "AI가 나한테 4점?!" 분노/웃음 → 댓글 논쟁 + 공유 폭발
- **적용법**: AI 기반 점수화는 어떤 카테고리든 논쟁 + 공유를 동시에 유발

#### D. SMV(Sexual Market Value) 계산기
- **개요**: 나이, 수입, 외모 자가평가, 키 등을 입력하면 "연애 시장에서의 가치"를 1~10점으로 산출하는 웹 도구. 레드필 커뮤니티에서 시작됐지만, "내 시장 가치가 몇 점?"이라는 궁금증이 일반 유저까지 끌어들였다.
- **바이럴 원리**: 혐오하면서도 안 볼 수 없는 콘텐츠 — 결과가 높으면 자랑, 낮으면 자조적 공유
- **적용법**: 표면 스펙이 아닌 다른 프레임(성격, 역량, 라이프스타일) 기반으로 변환하면 덜 공격적이면서 같은 효과

#### E. 아만다 (Amanda) — 외모 심사 소개팅앱
- **개요**: 가입 시 기존 회원 다수가 외모를 5점 만점으로 평가하고, 3.0점 이상이어야 가입 승인. 노골적 외모 심사제로 "나는 통과할 수 있을까?"라는 불안이 가입 자체를 바이럴 이벤트로 만들었다.
- **바이럴 원리**: 게이트키핑 → "통과/탈락" 구조가 도전 욕구 + 논란 동시에 유발. 논쟁 자체가 무료 마케팅
- **규모**: 누적 630만 회원. 가품 여성 계정 생성으로 2.2억원 과징금
- **적용법**: "심사제" 게이트키핑 — 통과해야 프리미엄 기능 오픈. 진입 장벽이 오히려 욕망을 키움

#### F. 골드스푼 — 자산 인증 소개팅앱
- **개요**: 남성은 소득 증빙서류(30대 기준 연 7,000만+), 여성은 기존 회원 외모 평가 3.6/5 이상을 통과해야 가입. 수락률 30%. "자산과 외모로 사람을 나누는 앱"이라는 논란이 오히려 가입 시도를 폭발시켰다.
- **바이럴 원리**: 노골적 계층화 + 엘리트 FOMO. 논란이 클수록 "나는 통과할까?" 호기심 증가
- **규모**: 14만 인증회원, 630만 누적 가입 시도
- **적용법**: 티어링(Gold/Silver/Bronze 등급) → 비교 + 상위 등급 열망 유발

### 🟠 FOMO / 희소성 ("지금 안 하면 놓친다")

> 사람은 "있는 것을 얻는 기쁨"보다 **"있던 것을 놓치는 고통"을 2배 더 크게 느낀다** (손실 회피).
> 시간 제한, 인원 제한, 초대 제한 — 어떤 형태든 **"지금 아니면 사라진다"**는 신호가 행동을 강제한다.

#### G. Thursday — 목요일에만 작동하는 데이팅앱
- **개요**: 매주 목요일 오전 6시~자정, 딱 24시간만 매칭과 채팅이 가능한 데이팅앱. 자정이 되면 모든 대화와 매칭이 삭제된다. "오늘 안 하면 다음 주 목요일까지 기다려야 한다"는 극단적 인공 희소성이 핵심.
- **바이럴 원리**: 주 1회 한정 → 목요일마다 SNS에서 "오늘 Thursday 했어?" 화제. 게릴라 마케팅(직원이 "목요일에만 데이트 가능" 현수막 들고 거리 돌아다니기)으로 19일 만에 £250만 투자 유치
- **규모**: 수백만 DL
- **적용법**: 특정 요일/시간에만 열리는 한정 기능 → 주기적 FOMO 생성

#### H. Bumble 24시간 타이머
- **개요**: 틴더와 달리 여성이 먼저 메시지를 보내야 대화가 시작되는 구조. 매칭 후 24시간 안에 메시지를 보내지 않으면 매칭이 영구 소멸한다. 화면에 표시되는 카운트다운 타이머가 행동을 강제한다.
- **바이럴 원리**: 손실 회피 + 시각적 긴급성. "이 사람을 영원히 놓칠 수 있다"는 압박이 즉각적 행동을 유도
- **규모**: 글로벌 4,000만+ 유저. 학술 논문 주제가 될 정도
- **적용법**: 시한부 결과/기회 — "OOO 창이 6시간 열려있습니다" 카운트다운

#### I. Raya / The League — 초대제 엘리트 앱
- **개요**: Raya는 셀럽·크리에이터 전용 데이팅앱으로, 기존 회원 추천 + 인스타 심사를 거쳐야 가입 가능. 대기자 250만 명, 수락률 8%. The League는 링크드인 인증 기반으로 대기자 2~3만 명. 벤 애플렉이 매칭 상대에게 거절당한 뒤 보낸 인스타 DM 영상이 틱톡에서 바이럴되기도 했다.
- **바이럴 원리**: "들어갈 수 없다"는 사실 자체가 바이럴. 배타성이 욕망을 키움
- **적용법**: "월 N명 한정" — 희소성이 서비스의 인지 가치를 끌어올림

#### J. Cuffing Season / Hot Girl Summer
- **개요**: 영미권에서 가을~겨울은 "커플링 시즌(Cuffing Season)"이라 불리며, 외로움을 피하기 위해 연인을 만드는 시기. 반대로 여름은 Megan Thee Stallion이 만든 "Hot Girl Summer" 밈과 함께 싱글 자유를 만끽하는 시기. 이 계절 심리가 데이팅앱 사용량을 겨울 30%↑로 끌어올린다.
- **바이럴 원리**: 계절 FOMO — "다들 커플 되는데 나만 혼자" 불안이 행동을 유발
- **적용법**: 시즌별 콘텐츠 — 계절 심리를 활용한 시의성 캠페인 발행

### 🟡 동질집단 / 친구 역학 ("친구들은 어떨까?")

> 혼자 하는 서비스는 "소비"에서 끝나지만, **친구와 함께 하는 서비스는 "놀이"가 된다.**
> 친구를 끌어들이는 구조가 내장되면, 유저 1명이 3~5명을 데려오는 바이럴 루프가 만들어진다.

#### K. Ship — 친구가 대신 스와이프하는 앱
- **개요**: 싱글 친구를 위해 3~5명이 "크루"를 만들어 대신 프로필을 스와이프해주는 데이팅앱. 그룹채팅에서 "이 사람 어때?"라고 토론한 뒤 매칭. 매치의 53%가 본인이 아닌 친구가 스와이프한 결과였다.
- **바이럴 원리**: 1명이 참여하면 3~5명이 자동 동반 + "친구가 골라줬다"는 사회적 증거
- **규모**: Match Group 내 가장 빠른 성장 (2022 종료)
- **적용법**: "친구가 봐주는 나의 OOO" — 친구들이 투표/평가하는 구조

#### L. "Are We Dating the Same Guy?" 페이스북 그룹
- **개요**: 여성들이 데이팅앱에서 만난 남성의 프로필 스크린샷을 올리고 "이 사람 아는 분?"이라고 물어보는 페이스북 비공개 그룹. 바람둥이·가스라이터 경고를 공유하는 일종의 보호 네트워크로 기능하며, 전세계 200개+ 도시에 그룹이 생겼다.
- **바이럴 원리**: 집단 지성 + 자기 보호 + 가십 — 세 가지 동기가 동시에 작동
- **규모**: 런던 그룹만 9만 명. WP/Vice/NBC 보도
- **적용법**: 프라이버시 문제 없는 방식으로 "그 사람의 진짜 성격/유형" 분석 기능

#### M. Locket Widget — 홈화면 친구 사진
- **개요**: 친구가 찍은 사진이 내 폰 홈화면 위젯에 실시간으로 뜨는 앱. 원래 대학생이 여자친구 생일 선물로 만든 개인 프로젝트였는데, 틱톡 영상 1개가 하루 500만 조회를 찍으면서 2주 만에 200만 가입이 폭발했다.
- **바이럴 원리**: 친밀감(친구 사진이 내 폰에) + FOMO(친구들은 다 쓰는데) → 자연스러운 확산
- **규모**: 9,000만+ DL, 일 900만 DAU, 100억+ 사진 공유. 2024년부터 흑자
- **적용법**: 홈화면 위젯 형태의 일일 콘텐츠 + 친구와 결과 비교 기능

#### N. Hater App — 같이 싫어하는 것으로 매칭
- **개요**: "좋아하는 것"이 아니라 "싫어하는 것"으로 매칭하는 역발상 데이팅앱. 3,000개 이상 주제(피클, 비 오는 날, 느린 와이파이 등)에 대해 싫어요를 누르면 같은 것을 싫어하는 사람과 연결. Shark Tank에서 마크 큐반이 $200K 투자.
- **바이럴 원리**: 심리학 연구에 따르면 부정적 유대가 긍정적 유대보다 강함. "같이 싫어하는 것"이 "같이 좋아하는 것"보다 친밀감 형성에 효과적. 역발상 자체가 뉴스거리
- **규모**: 출시 50만 DL (2019 종료)
- **적용법**: 역발상 매칭 — "공통으로 싫어할 것" 분석은 공감력이 높음

#### O. 이상형 월드컵 (PIKU)
- **개요**: 16~32명의 후보 중 둘씩 비교해서 하나를 고르는 토너먼트 양자택일 게임. 2009년 플래시 게임으로 시작해, 2017년 PIKU 앱으로 폭발. "너는 누구 골랐어?"라는 대화가 자연스럽게 확산을 만든다.
- **바이럴 원리**: 이진 선택 = 인지 부담 제로(생각할 필요 없이 직감으로 터치) + 결과 비교가 대화 소재
- **규모**: 인터넷 방송인 채택 → 2020년대 콘텐츠 필수 포맷
- **적용법**: 사진 대신 성격/유형 프로필 대결 → 이상형 유형 도출

### 🟢 수치/취약성이 바이럴 연료 ("공감+자조")

> 완벽한 모습보다 **솔직한 약점 고백**이 더 넓게 퍼진다.
> "나만 이런 줄 알았는데 다들 그렇구나" — 이 공감의 순간이 공유 버튼을 누르게 한다.
> 특히 실패를 유머로 포장하면 공유 저항이 거의 제로가 된다.

#### P. "He's a 10 But..." 트렌드
- **개요**: "외모가 10점인 사람인데, [단점]이 있으면 몇 점?" 형식의 틱톡/트위터 밈. 예: "He's a 10 but 답장이 3일 후에 옴 → 몇 점?" 모든 사람이 자기만의 버전을 만들 수 있어 무한 리믹스가 가능하다.
- **바이럴 원리**: 가치관이 드러나는 판단 + 정답이 없어서 논쟁이 끝나지 않음 + 누구나 참여 가능
- **규모**: 틱톡 #hesa10but 1억+ 조회, 트위터 실검 1위
- **적용법**: "OOO 기준 10점인데..." — 항목별 점수를 분리해서 종합 점수 산출하는 형태

#### Q. "The Ick" 트렌드
- **개요**: 상대에게 호감이 있다가, 아주 사소한 행동(물 마시는 방식, 달리는 자세 등) 하나에 갑자기 매력이 사라지는 순간을 "Ick"이라 부르며 고백하는 SNS 트렌드. 2023년 Dictionary.com에 공식 등재될 정도로 Gen Z 필수 연애 용어가 되었다.
- **바이럴 원리**: 비합리적이라서 웃기고 + "나도 그래!" 공감 + 끝없이 릴레이 가능
- **규모**: @wyszkay Ick 리스트 30만+ 좋아요
- **적용법**: 프레임워크 기반 "당신의 ICK 예측" — 성격/유형의 불균형 요소를 기피 행동으로 변환

#### R. Orange Peel Theory (오렌지 껍질 테스트)
- **개요**: 파트너에게 "오렌지 껍질 까줄 수 있어?"라고 부탁하고 반응을 촬영하는 틱톡 트렌드. Gottman 연구의 "관계 신호(bid for connection)"를 일상 실험으로 만든 것. 귀찮아하면 불합격, 흔쾌히 까주면 합격.
- **바이럴 원리**: "그가 진심인지" 불안 + 누구나 즉시 실행 가능 + 합격/불합격의 드라마
- **규모**: #OrangePeelTheory 3,440만 조회
- **적용법**: 간단한 "진심 테스트" 포맷 — 최소 입력 → 헌신/봉사 성향 퍼센티지 출력

#### S. NGL App (익명 질문)
- **개요**: 인스타 스토리에 "나에게 아무거나 물어봐"라는 익명 질문 링크를 올리면, 팔로워들이 익명으로 질문이나 고백을 보내는 앱. "사람들이 나를 진짜 어떻게 생각하지?"라는 궁금증을 건드려서 한 달에 앱스토어 1위를 찍었다.
- **바이럴 원리**: 익명성이 솔직함을 유발 + 스토리에 올리면 팔로워 전원에게 노출 → 체인 반응
- **규모**: **한 달 727만 DL** (앱스토어 1위). 1,500만 DL, $240만 매출 (FTC $500만 과징금)
- **적용법**: "나에 대해 아무거나 물어봐" 형태의 익명 링크 → AI가 프레임워크 기반으로 답변

#### T. Whisper App (익명 고백)
- **개요**: 배경 이미지 위에 익명으로 비밀을 고백하는 앱. "사실 나 전 남친이 아직 좋아" 같은 연애/이별/짝사랑 고백이 주력 콘텐츠. PostSecret(엽서 비밀 프로젝트)의 모바일 버전이라 할 수 있다.
- **바이럴 원리**: 익명이라 진짜 속마음이 나옴 → 공감 폭발 → 스크린샷 공유
- **규모**: 월 30억 PV, MAU 3,000만. $2억 밸류에이션 (2025.7 종료)
- **적용법**: 유형별 익명 고백 수집 → 패턴 분석 → 콘텐츠화

### 🔵 상태 과시 / 질투 유발

> SNS의 본질은 **"나의 삶이 이렇게 좋다"를 보여주는 것**이다.
> 커플 자랑, 마일스톤 달성, 특별한 유형 카드 — 과시할 수 있는 "소재"를 만들어주면 유저가 알아서 퍼뜨린다.
> 보는 사람은 부러움을 느끼고, 그 부러움이 서비스 유입으로 전환된다.

#### U. 한국 100일 기념일 문화
- **개요**: 한국 커플은 사귄 지 100일/200일/300일/1000일마다 기념일을 챙기며, 케이크·선물·SNS 포스팅을 한다. 100일마다 커플 콘텐츠가 SNS에 파도처럼 밀려오고, 싱글들은 FOMO를 느낀다.
- **바이럴 원리**: 커플의 과시 → 싱글의 FOMO → 관계/연애 서비스 수요 증가
- **적용법**: 마일스톤 예측 콘텐츠 — "100일에 권태기", "200일에 황금기" 등 시점별 인사이트

#### V. Between (비트윈) 커플앱
- **개요**: 커플 전용 앱으로 공유 캘린더, 사진 앨범, D-day 카운터, 커플 스티커 등을 제공. 앱을 쓰고 있다는 것 자체가 "나 연애 중" 시그널이 되어, 주변 친구들에게 자연스럽게 전파된다.
- **바이럴 원리**: 앱 보유 = 관계 과시. 마일스톤(100일, 1년)마다 자동 생성되는 기념 카드 → SNS 공유
- **적용법**: 데일리 콘텐츠 — "오늘의 궁합 에너지 89%" 같은 일일 리포트 형태

#### W. #CoupleGoals 콘텐츠
- **개요**: 틱톡/인스타에서 수십억 조회를 기록한 커플 콘텐츠 장르. Matt & Abby(500만 팔로워) 같은 커플 인플루언서가 일상을 공유하면, 보는 사람들이 "나도 저렇게 되고 싶다"는 열망을 느낀다.
- **바이럴 원리**: 부러움 + 열망 → "이상적 관계"에 대한 판타지 소비 → 관계 관련 서비스 수요
- **적용법**: 아키타입 카드 — "천생연분 커플", "불꽃 커플" 같은 유형 카드 제공 → #couplegoals 포스팅 유도

### 🟣 국내 추가 케이스

> 한국 시장만의 특수한 바이럴 패턴. **카카오톡 공유 최적화**와 **테스트 플랫폼 유통**이 핵심 변수.

#### X. 푸망 (Poomang) — 심리테스트 플랫폼
- **개요**: "심리테스트계의 넷플릭스"를 표방하는 플랫폼. 다양한 심리테스트를 카톡 공유에 최적화된 형태로 제공. 6개월 만에 683만 유저, 누적 7,500만 테스트 플레이. 핵심 데이터: **광고 티가 안 나는 테스트가 광고성 테스트보다 2배 더 많이 플레이됨**.
- **규모**: MAU 224만. 시장 점유율 20~30%
- **적용법**: 외부 테스트 플랫폼에 유통할 때 브랜딩을 최소화하고 "순수한 재미"로 포장해야 플레이 수가 2배

#### Y. 연애의 과학 — 카톡 분석 앱
- **개요**: 카카오톡 대화를 내보내기하면 AI가 상대의 감정 상태, 호감도, 대화 패턴을 분석해주는 앱. "쟤가 날 좋아해?"라는 질문에 데이터로 답해준다. 77,050명의 대화를 분석했고, 유료 카톡 분석이 베스트셀러 기능. 다만 수집한 대화 데이터로 AI 챗봇 "이루다"를 학습시켜 개인정보 논란 + 과징금을 받은 전력이 있다.
- **바이럴 원리**: "쟤가 날 좋아할까?"는 인류 최대의 궁금증. 실제 대화 데이터 기반이라 신뢰감
- **적용법**: 프라이버시 이슈 없는 방식(실제 대화 수집 없이)으로 소통 패턴 예측 기능 설계

#### Z. "전 연인 복기" (Rate My Ex 변형)
- **개요**: 틱톡에서 전 연인들을 순위 매기거나 점수를 붙이는 트렌드. "엄마가 내 전남친들 평가해줌", "내 전 연인 티어 리스트" 같은 변형이 계속 파생된다. 관음증 + 솔직한 취약성 + 유머가 결합된 포맷.
- **바이럴 원리**: 아픈 과거를 웃기게 정리하는 카타르시스 + "다른 사람 연애사 구경하고 싶다" 관음증
- **적용법**: 과거 관계 분석 — 아픈 역사를 객관적 프레임워크(유형, 점수, 타임라인)로 재해석

| **비교/랭킹** | "나는 몇 점?" | 점수 자랑 or 자조 | Tinder ELO, AI 매력점수 | 퍼센타일/등급 카드 출력 |

## 12. 바이럴 기능 설계: 범용 티어 분류

### S-Tier: 강제 공유 루프가 내장된 기능

#### ① "나를 얼마나 아는지" 퀴즈
- **구조**: 자사 프레임워크 기반 10개 질문 자동 생성 → 고유 URL → 친구에게 공유 → 맞추기
- **바이럴 포인트**: 제품 자체가 공유 없이 작동 불가 (BuddyMeter 공식)
- **유료 전환**: 퀴즈 결과 후 "진짜 내 분석 보기" CTA
- **개발 난이도**: 중간 (질문 생성 AI + 고유 URL + 점수 시스템)

#### ② 초저마찰 궁합/매칭 카드
- **구조**: 최소 입력(이름/생년월일 등) 2개만 → 퍼센티지 + 예쁜 카드 → 공유
- **바이럴 포인트**: 이름궁합의 공식. 3초 입력 → 즉시 결과 → "우리 87%래!"
- **유료 전환**: "상세 분석 보기" CTA
- **개발 난이도**: 낮음 (기존 기능 + 공유 카드 디자인)

### A-Tier: 정체성 표현 + 공유 카드

#### ③ 퍼스널 컬러/유형 카드 (Ktestone 공식)
- **구조**: 자사 데이터/프레임워크 기반 → "나의 컬러/유형" + 성격 요약 → 예쁜 컬러 카드
- **바이럴 포인트**: Ktestone이 4.9억 조회를 만든 바로 그 공식. **카드가 예뻐야 공유함**
- **유료 전환**: "내 유형으로 보는 상세 분석" CTA
- **개발 난이도**: 낮음

#### ④ 성격/체질 유형 카드
- **구조**: 프레임워크 기반 유형 분류 ("불꽃형", "바다형" 등) + 잘 맞는 유형
- **바이럴 포인트**: MBTI 16유형처럼 "너 뭐 유형?" 대화 유발
- **유료 전환**: "나의 유형 상세 분석" CTA
- **개발 난이도**: 중간 (유형 체계 설계 + 카드 디자인 + 매칭 매트릭스)

### B-Tier: 시즌/이벤트 연동

#### ⑤ "OOO Wrapped" 리포트
- **원형**: 스포티파이가 매년 12월에 발행하는 "Spotify Wrapped" — 올해 가장 많이 들은 곡, 아티스트, 장르를 6~10장짜리 세로 카드 시퀀스로 보여주고 인스타 스토리에 공유하게 만드는 포맷. 전세계적으로 12월마다 SNS를 도배하며, Dating Wrapped(올해 연애 요약), 독서 Wrapped 등 수많은 패러디가 파생됐다.
- **구조**: 자사 서비스의 데이터를 6장짜리 카드 시퀀스로 요약 — 카테고리별 핵심 키워드/하이라이트 시점/주의할 시점/총평. 각 카드가 인스타 스토리 9:16 비율로 개별 공유 가능.
- **바이럴 포인트**: Wrapped 포맷은 이미 글로벌 검증 완료. "나의 1년을 한 장으로 정리"하는 정체성 표현 + 연말/반기 시즌 한정 희소성이 동시에 작동
- **유료 전환**: "하반기 상세 분석" CTA
- **개발 난이도**: 중간 (6장 카드 시퀀스 + 스와이프 애니메이션)

#### ⑥ 밸런스 게임 (자사 프레임워크 버전)
- **구조**: 유형별 맞춤 밸런스 질문 → 답변 후 "프레임워크로 본 너의 선택 해석"
- **바이럴 포인트**: 논쟁 유발 + 자사만의 해석 추가 → 기존 밸런스 게임과 차별화
- **유료 전환**: "성향 풀분석" CTA
- **개발 난이도**: 낮음

## 13. 핵심 인사이트 (잊지 말 것)

1. **"검색량 = 관심도" 공식이 붕괴 중** — AI 검색 전환으로 전통 검색량 전반 하락. 관심도 측정은 SNS 언급량 + 유튜브 조회수 + 커뮤니티 활성도를 크로스 체크해야 함.

2. **연애 콘텐츠가 모든 플랫폼에서 압도적** — 유튜브 기준 '성공'의 23배. Threads/디시인사이드에서도 4~8배. 관계/연애 카테고리는 분야 무관 최강의 바이럴 소재.

3. **SNS 인기글의 본질 = 자기진단 + 미래예측** — "나는 왜 이런가" + "앞으로 어떻게 될까". 이 두 질문에 답해주는 서비스는 자연스러운 바이럴 구조를 갖는다.

4. **바이럴 기능의 핵심은 "예쁜 결과 카드 + 공유 버튼"** — W컨셉 2주 18만, 스포티파이 Wrapped 글로벌 바이럴. 복잡한 기능보다 **공유하고 싶은 결과 이미지**가 핵심.

5. **속도가 곧 바이럴** — 토스 두쫀쿠맵 하루 만에 출시. 트렌드 감지 후 24시간 이내 콘텐츠 발행 체계 구축 필요.

6. **강제 2인 루프가 최강의 바이럴** — Friendship Quiz(틱톡 5억 조회), Paired(800만 DL). 제품이 공유 없이 작동 불가한 구조가 가장 강력한 바이럴.

7. **Ktestone이 가장 직접적 벤치마크** — 한국 발 글로벌 바이럴, 틱톡 4.9억 조회. 핵심은 MBTI를 "컬러"로 리패키징한 것. 기존 프레임워크를 시각적으로 리패키징하면 동일 효과.

8. **AI가 대체할 수 없는 것에 집중** — 일반적인 조언은 ChatGPT가 대체. **고유 데이터 기반 객관적 진단**은 대체 불가.

9. **숫자를 주면 공유한다** — Photofeeler, ELO, SMV, AI 매력점수. 퍼센타일/점수 형태의 결과는 자랑이든 자조든 반드시 공유됨.

10. **논란 자체가 마케팅** — 아만다(외모 심사), 골드스푼(자산 인증). 논란이 클수록 미디어 노출 폭발. 적절한 티어링/등급화는 논란 유발 가능.

11. **광고 티 나면 플레이 수 절반** — 푸망 데이터: 브랜딩 최소화한 테스트가 2배 더 플레이됨. 테스트/퀴즈는 "순수한 재미"로 포장해야 함.

12. **자조 유머가 가장 넓게 퍼진다** — He's a 10 but(1억 조회), Dating Wrapped(1,030만). 실패를 웃기게 포장하는 포맷이 가장 공유 저항이 낮음.

## 14. 그로스 실험 프레임워크 (A/B 테스트)

> 바이럴 기능의 효과를 "감"이 아닌 **통계 검정**으로 판단하라.

### 통계 검정 방법

| 방법 | 용도 |
|------|------|
| **Mann-Whitney U** (비모수) | 정규분포 가정 없이, 작은 표본에서도 유의성 검정 |
| **부트스트랩 CI** (1000회 리샘플링) | 실제 개선 폭의 95% 신뢰구간 추정 |

### 의사결정 임계값

| 판정 | 조건 | 액션 |
|------|------|------|
| **승인** | p < 0.05 AND 개선도 ≥ 15% | 정식 적용, 플레이북 등록 |
| **추세** | p < 0.10 AND 표본 ≥ 15개 | 데이터 수집 계속 |
| **폐기** | p < 0.05이고 기존안 우수 | 실험 종료 |
| **진행중** | 위 조건 불충족 | 계속 |

**핵심**: 통계적 유의성(p < 0.05)만으로 부족 — **실무적 의의(≥15% 개선)**도 동시 충족해야 승인
